# SSO Cendekia Documentation Hub

This guide is intended for end users to be able to log in to applications connected to Scholar SSO quickly and securely.

## What is Scholar SSO?

Scholar SSO is a Single Sign-On (SSO) system that allows users to access multiple applications and services with one unified account. With Scholar SSO, you don't need to create and remember separate passwords for each application.

### Main Function

- **Unified Authentication**: Sign in once to access all integrated applications
- **Guaranteed Security**: Layered security system to protect user data
- **Ease of Access**: Social login via Google or Microsoft
- **Centralized Account Management**: Manage profiles and access permissions from one place

## What You Will Do

- Log in with email and password
- Register a new account
- Login with Google or Microsoft
- Reset password when forgotten
- Approve application access (consent)

## Before Starting

- Make sure you have received the application link from the institution/company.
- Use the latest browser (Chrome, Edge, Firefox, Safari).
- Prepare an active email for account verification and password reset.
