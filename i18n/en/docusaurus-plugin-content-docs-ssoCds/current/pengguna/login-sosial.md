# Login with Google or Microsoft

Scholar SSO supports social login to make the login process faster.

## Social Login Steps

1. On the login page, click Continue with Google or Continue with Microsoft.
2. Select the account you want to use.
3. Approve the permission request from the provider.
4. You will return to the original application in a logged in state.

## Why are you asking for permission?

Permissions are used to retrieve basic profile data, for example:

- Name
- E-mail
- Provider account identity

## Frequently Occurring Problems

- The provider account does not have an active email.
- You closed the popup before the process was complete.
- The browser blocks third-party cookies/session.

Quick solution: repeat login, then use private/incognito mode if necessary.
