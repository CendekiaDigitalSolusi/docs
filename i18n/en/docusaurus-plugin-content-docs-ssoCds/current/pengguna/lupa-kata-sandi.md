# Forgot Password

If you forget your password, use the following reset process.

1. On the login page, click [Forgot Password](https://accounts.cendekiadigitalsolusi.co.id/forgot-password).
2. Enter your account email.
3. Click Send Reset Link.
4. Open the email and click the reset link.
5. Enter the new password, then save it.
6. Login again with the new password.

![Password reset flow](/img/sso-cendekia/reset-password-flow.svg)

## Vital Records

- The reset link is only valid temporarily.
- If the link expires, repeat the Forgot Password process.
- After reset, the login session on the old device can end automatically.
