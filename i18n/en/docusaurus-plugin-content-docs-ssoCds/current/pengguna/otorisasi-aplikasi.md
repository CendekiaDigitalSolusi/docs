# Application Access Approval (Consent)

The first time you use a particular app, you may see an access approval page.

![Example of consent screen](/img/sso-cendekia/consent-screen.svg)

## What does it mean?

The application asks for permission to access your account data according to the requested scope.

Examples of common data:

- `openid`: the user's unique identity
- `email`: email address
- `profile`: base profile name

## What to Check Before Clicking Allow

- The application name is correct
- Application domain according to what you are familiar with
- The type of data requested is reasonable

If in doubt, click Deny then contact the app admin.
