# How to Enter (Login)

## Login with Email and Password

![Login Page](/img/sso-cendekia/login-page.png)
1. Go to the application login page. (Example: https://gradynex.com/login)
2. Click the Sign in with SSO button or you will be automatically directed to the SSO login page.
3. Fill in your email and password.
4. Click Sign in.
5. If successful, you will return to the original application.

## If the Access Approval Page Appears

1. Check the name of the application requesting access.
2. Check the requested data (e.g. email and basic profile).
3. Click Allow to continue.

## Failed to Login?

- Make sure the email and password are correct.
- Check Caps Lock keyboard.
- Use the Forgot Password menu if necessary.
- If your account is locked due to multiple failed attempts, wait a few minutes and try again.
