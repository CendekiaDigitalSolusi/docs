# How to Register a New Account

## Registration Steps

1. On the SSO login page, click [List](https://accounts.cendekiadigitalsolusi.co.id/register).
2. Fill in the requested data (name, email, password).
3. Click Create Account.
4. Check your email inbox for verification.
5. Click the verification link, then log in again.

## Secure Password Rules

Use a password containing a combination:

- Uppercase and lowercase letters
- Number
- Symbol
- Minimum length 8-12 characters

Example of a safe pattern: combine 3 random words + number + symbol.

## If the verification email doesn't arrive

- Check the Spam/Junk folder.
- Make sure the email entered is correct.
- Wait 1-3 minutes and refresh inbox.
