# User FAQ

## I have logged in but returned to the login page again

- Check the time/device clock, make sure it is accurate.
- Make sure browser cookies are not blocked.
- Try logging out then logging in again.

## My account is locked

This usually occurs because repeated failed login attempts. Wait a few minutes then try again, or reset the password.

## I didn't receive a verification or reset email

- Check the Spam/Junk folder.
- Make sure the email is correct.
- If it's still not there, contact the admin to check the email service.

## When you click Allow, a redirect_uri error appears

This is usually an application configuration issue (not user error). Send error screenshots to the application admin team.

## When should I contact admin?

Contact admin if:

- repeated account locks
- The verification/reset email never arrived
- access is denied even though the account is active
- a technical error appears after logging in
