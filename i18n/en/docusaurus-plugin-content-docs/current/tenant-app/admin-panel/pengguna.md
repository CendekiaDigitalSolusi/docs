# User

The **Users** page is used to manage accounts that can log into the Tenant system.

Here you can:
- Adding a new user account
- Set user roles
- Quickly search for specific accounts
- Changing user data
- Delete account

## How to Access

1. Entered into [Admin Dashboard](./dashboard-admin.md)
2. Open menu **Admin → Users**

## User Page Elements

### Toolbar on Top of Table

- **Add New Button**: Creates a new user account
- **Action Dropdown**: Performs bulk actions for checked items
- **Column Dropdown**: Set which columns are displayed
- **Number of Rows Dropdown**: Sets how much data is displayed per page (example: 10, 25, 50)

### User Table

The table displays the following columns:

| Kolom | Description |
|-------|-----------|
| **Name** | User's full name |
| **Username** | Username to login |
| **Email** | User email |
| **Role** | Role/user access rights |
| **Action** | Edit/Delete button for each user |

### Quick Filter

Each column header has filters for quick search:
- Filter **Name**
- Filter **Username**
- Filter **Email**
- Filter **Role** (dropdown)

## Adding New Users

1. Click the **Add New** button
2. Fill in the form with user data:
   - Full name
   - Username (to log in)
   - E-mail
   - Passwords
   - Confirm password
   - Select Role
3. Click **Save**

## Changing User Data

1. Search for the user you want to convert using filters or search
2. Click the **Edit** button (pencil icon) in the user row
3. Change the required data
4. Click **Save**

## Deleting Users

1. Search for the user you want to delete
2. Click the **Delete** button (trash icon) in the user row
3. Confirm deletion

:::warning
Be careful when deleting users, as this action cannot be undone. Make sure the user is no longer needed.
:::
