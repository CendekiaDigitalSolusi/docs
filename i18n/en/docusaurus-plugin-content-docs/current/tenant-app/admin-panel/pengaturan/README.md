# Arrangement

The Settings module allows you to configure the system according to your school's needs.

## Settings Components

This module includes:

- **[General Settings](./pengaturan-umum.md)** - Basic website configuration and SEO
- **[School Profile](./profil-sekolah.md)** - School identity and profile data
- **[Email Server Settings](./pengaturan-server-email.md)** - Automatic email configuration
- **Blog** - Custom settings for the blog module
  - [Reading Settings](./blog/pengaturan-membaca.md)
  - [Discussion Settings](./blog/pengaturan-diskusi.md)

## Quick Guide

To set up the system, study the above topics according to your needs.

### For Initial Setup
1. Start with [General Settings](./pengaturan-umum.md)
2. Continue to [School Profile](./profil-sekolah.md)
3. Setup [Email Server](./pengaturan-server-email.md) if necessary

### For Blog Configuration
- Look [Blog Settings](./blog/README.md)
