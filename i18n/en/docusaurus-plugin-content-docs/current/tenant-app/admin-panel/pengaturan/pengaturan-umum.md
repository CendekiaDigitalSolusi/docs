# General Settings

**General Settings** is the basic configuration center for the school website. Here you can set brand identity, basic SEO, time zone, and security (reCAPTCHA).

## How to Access

1. Entered into [Admin Dashboard](../dashboard-admin.md)
2. Open menu **Admin → Settings → General Settings**

## General Settings Components

### Visual Identity (Logo & Favicon)

In this section there are 4 types of logos and favicon:

**Light Logo:**
- Used when the header background is bright
- Click **Upload** to replace

**Dark Logo:**
- Used when the header background is dark
- Click **Upload** to replace

**Favicon:**
- A small icon that appears in the browser tab
- Usually the size is 32x32 or 64x64 pixels

:::tip
Use SVG or PNG format for best results. The logo will be displayed at various sizes, make sure it remains clearly visible.
:::

### SEO Settings (Meta)

SEO here is "foundational". The important thing is that it is neat and relevant to the website topic.

#### Meta Description
- Fill in a brief description of your school
- Usually appears as a preview in search engines
- Suggestion: 1-2 sentences explaining the school + main advantages
- Avoid keyword stuffing (excessive use of keywords)

**Example format:**
> Digital School provides school profile information, programs and the latest news. Access academic and online learning services easily.

#### Meta Keywords
- Complete a list of keywords that describe the school
- Separate with commas
- Example:`Sekolah Digital, Profil Sekolah, PPDB, Berita Sekolah, LMS, Akademik`

:::info
Meta keywords don't always have a big influence in modern SEO, but still fill them in enough for metadata consistency.
:::

### Time Zone Settings

- Select the time zone according to your school location
- Use it to synchronize time on the system

### Security Settings

#### reCAPTCHA
- If enabled, visitors must verify reCAPTCHA when logging in/submitting certain forms
- Helps prevent bots and unauthorized access

## Tips

- **Update logo** - Make sure your logo is high quality
- **SEO foundation** - Fill in an accurate and interesting meta description
- **Precise timezone** - Select the correct time zone for system time accuracy
- **Periodic review** - Update settings if there are changes to the school profile
