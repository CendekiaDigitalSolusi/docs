# Image Slider

The **Image Slider** page manages a list of slider images that appear on public pages (for example on the school website homepage).

Each slider item usually contains an image, title, and description. Some schools also add links/targets when the slider is clicked.

## How to Access

1. Entered into [Admin Dashboard](../dashboard-admin.md)
2. Open menu **Admin → Blog → Image Slider**

## Image Slider Page Elements

**Toolbar above the table:**
- **"Add New" button**: Creates a new slider
- **"Search" box**: Searches the slider by title
- **Action Dropdown**: Bulk actions for selected sliders
- **Column Dropdown**: Set which columns are displayed
- **Number of rows dropdown**: Set how much data per page

**Table column:**
- **Title** - Slider name/label
- **Description** - Short text that appears in the slider
- **Action** - Edit/Delete button

## Creating a New Slider

1. Click the **Add New** button
2. Fill out the form:
   - **Title**: The name of the slider you want to display
   - **Description**: Short text (optional)
   - **Image**: Upload an image for the slider
   - **Link/Target**: Link that opens when the slider is clicked (optional)
3. Click **Save**

## Changing Sliders

1. Find the slider you want to change
2. Click the **Edit** button on the slider row
3. Change the title/description/image as needed
4. Click **Save**

## Removing Sliders

1. Find the slider you want to delete
2. Click the **Delete** button on the slider row
3. Confirm deletion

## Tips

- **Use short titles** - Titles should be easy to read and not too long
- **Make sure the image quality is good** - Avoid images that are blurry or broken when displayed
- **Optimal image size** - Use a resolution appropriate for website performance (usually 1920x600px or 16:10 ratio)
- **Order of importance** - The first slider is the one that visitors see the most
- **Design consistency** - Use a consistent design style for all sliders
- **Periodic updates** - Change the slider regularly so that the website looks fresh

## Troubleshooting

**Image not showing:**
- Make sure the image format is supported (JPG, PNG, WebP)
- Check the image file size (don't make it too big)
- Try re-uploading the image

**Slider does not appear on website:**
- Check whether the slider module is activated
- Make sure there is at least 1 slider saved
- Clear website cache if necessary
