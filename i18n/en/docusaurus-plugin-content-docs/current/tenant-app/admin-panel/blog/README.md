# Blogs & News

The Blog module allows you to manage and publish content to the public pages of the school website.

## Blog Components

The blog module consists of several parts that you can manage:

- **[News](./berita.md)** - Date based dynamic content
- **[Page](./halaman.md)** - Evergreen static content
- **[Category](./kategori.md)** - Content grouping
- **[Tags](./tag.md)** - Additional labels for content
- **[Image Slider](./slider-gambar.md)** - Banner/carousel on homepage
- **[Comment](./komentar.md)** - Reader comments moderation

## Quick Guide

To start creating content, study the topics above according to the type of content you want to create.

### For News/Announcements
- See guide [News](./berita.md)

### For Static Content
- See guide [Page](./halaman.md)

### For Blog Settings
- Open [Blog Settings](../pengaturan/blog/README.md)
