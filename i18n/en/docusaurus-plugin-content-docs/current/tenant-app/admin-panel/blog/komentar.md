# Comment

Comments is where you manage visitor responses to **News** or **Pages**.

This feature is useful for:
- Keep discussions healthy and constructive
- Delete spam and irrelevant comments
- Monitor input and feedback from readers

## How to Access

1. Entered into [Admin Dashboard](../dashboard-admin.md)
2. Open menu **Admin → Blog → Comments**

## Comments Page Elements

**Toolbar above the table:**
- **"Search" box**: Search/filter comments
- **Action Dropdown**: Bulk action for selected comments
- **Column Dropdown**: Set which columns are displayed
- **Number of rows dropdown**: Set how much data per page

**Table column:**
- **Author** - The name of the person who wrote the comment
- **Comments** - Comment content
- **Created** - The time the comment was created
- **Action** - Buttons for approval, reject, or delete

## Moderated Work Pattern

### Routine Checkup

1. Open the **Comments** menu
2. Use the **Search** feature for sensitive keywords (example: links, promos, harsh words)
3. Read the latest comments from the **Created** column
4. Take the required action using the **Action** column

### Cleaning Spam

1. Filter with **Search** for spam patterns (example:`http`, `bit.ly`, `promo`)
2. Check some comments that are considered spam (if the checkbox is available)
3. Use the **Action** dropdown for the bulk delete action

## Moderation Actions

### Approve Comments
- Approved comments will appear on the public page
- Other readers can see the comments

### Reject Comments
- Reject comments without deleting them
- Comments will not appear publicly but will remain recorded

### Deleting Comments
- Permanently delete comments from the system
- Cannot be undone once deleted

## Moderation Tips

- **Check regularly** - Moderate comments daily if possible
- **Don't be quick to make decisions** - Read carefully before taking action
- **Consistent with the rules** - Use the same standards for all comments
- **Be professional** - Keep the tone of communication in moderation
- **Block spam automatically** - Make use of spam filters where available
