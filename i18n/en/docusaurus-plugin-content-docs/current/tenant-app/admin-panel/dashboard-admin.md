# Admin Dashboard

Admin Dashboard is the main page in the admin panel. Here you can see a summary of important information and quick access to school management features.

## How to Access the Admin Dashboard

1. [Enter](../autentikasi/masuk.md) to the system with an administrator account
2. You will see the user dashboard
3. Click the **Admin Panel** menu to enter the admin dashboard
4. Now you are in the Admin Dashboard

:::info
**You don't see the Admin Panel menu?** That means your account doesn't have access to the Admin Panel. Contact the school's primary administrator to request access.
:::

## Admin Dashboard Component

### Menu
The list of features that can be accessed according to your role/access rights includes:
- User Management
- System Settings
- Blog Management
- Appearance/Theme
- And other features according to the active module

### Data Summary
Cards display important statistical information and status of your school system.

:::warning
The features on the admin dashboard can change according to the modules activated or the system version you are using.
:::
