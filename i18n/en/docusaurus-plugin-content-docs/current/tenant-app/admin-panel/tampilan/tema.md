# Theme

The **Theme** page is used to choose the appearance/design of the school website. A theme manages the layout, visual components, colors, and overall style of a website.

## Theme Features

With this page, you can:
- Select a theme from the list of available themes
- Change the active theme
- Importing a new theme

:::warning
**Caution:** Changing themes can change the position of menus, sliders, and page appearance. Make changes during off-hours and check the public page afterward to make sure everything looks good.
:::

## How to Access

1. Entered into [Admin Dashboard](../dashboard-admin.md)
2. Open menu **Admin → Appearance → Themes**

## Theme Page Elements

This page displays the available theme cards.

**Each theme card typically features:**
- **Preview image** - A preview view of the theme
- **Theme Name** - Theme name (example: "Blue Frost", "School Vision")
- **Options Menu** - Triple dot icon for additional actions

**At the top right there is a button:**
- **Import Theme** - To import a new theme from a file

## Selecting/Activating Theme

The most common ways to change themes:

1. Determine the theme you want to use from the list provided
2. Click the **three dots** menu on the theme card
3. Select the **Enable** or **Use** option (button name may differ)
4. Open the public page of the website to make sure the display has changed

:::info
If the theme appears unchanged after activation, try a hard refresh in the browser by pressing **Ctrl+F5** (Windows) or **Cmd+Shift+R** (Mac).
:::

## Importing a New Theme

If you have a new theme file:

1. Click the **Import Theme** button at the top right
2. Select the theme file you want to import
3. Wait for the import process to complete
4. The new theme will appear in the themes list

## Tips

- **Test in staging first** - If there is a testing environment, try the new theme there first
- **Backup old themes** - Before deleting a theme, make sure you have a backup in case you need it again
- **Check responsive** - Open new themes on various devices (desktop, tablet, mobile)
- **Minimize custom CSS** - If custom CSS is necessary, document it for ease of maintenance

## Troubleshooting

**Theme does not change after activation:**
- Try hard refreshing the browser (Ctrl+F5)
- Clear browser cache
- Try in another browser

**Theme does not appear perfect:**
- Check the compatibility of the theme version with the system version
- See browser console for error message
- Contact support if the theme is damaged
