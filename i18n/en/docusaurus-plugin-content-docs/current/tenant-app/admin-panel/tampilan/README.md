# Appearance / Theme

The Appearance module allows you to manage the visuals and layout of the school website.

## View Components

This module includes:

- **[Menu](./menu.md)** - Manage website navigation
- **[Theme](./tema.md)** - Select and change the appearance of the website
- **[Module](./modul.md)** - Enable/disable features

To change the appearance of your website, study the topics above according to your needs.

### To Change Website Theme/Colors
- See guide [Theme](./tema.md)

### To Manage Features Displayed
- See guide [Module](./modul.md)
