# Menu

The **Menu** page is used to manage school website navigation. You can manage menus for **Header** and **Footer** (or other available variants).

Think of this page as a “road map” for your website visitors.

## How to Access

1. Entered into [Admin Dashboard](../dashboard-admin.md)
2. Open menu **Admin → Appearance → Menu**

## Main Mode

### Mode 1: Menu Structure
The **Menu Structure** tab is used to organize the order and hierarchy of menu items.

### Mode 2: Manage Menu
The **Manage Menu** tab is used to manage menu item lists (create/edit/delete).

:::info
**Work Order:** Usually you create menu items first in **Manage Menu**, then arrange their positions in **Menu Structure**.
:::

## Menu Page Components

### Left Panel: Select Menu

In the left pane, select the menu set you want to edit. Frequently available menus:

- **Header Menu** - Main navigation at the top of the website
- **Footer Menu** - Link at the bottom of the website

**Buttons on left panel:**
- **Add New**: Creates a new menu set
- **Change**: Change the menu name/configuration

### Right Panel: Manage Structure & Items

Depending on the tab selected, you can:

**In the Manage Menu Tab:**
- Create a new menu item
- Edit existing menu items
- Delete menu items

**In the Menu Structure Tab:**
- Set the order of items (drag & drop)
- Arrange hierarchies/submenus
- Hide certain items

## Adding Menu Items

Menu items can come from several sources:

### Manual Link
- Manual link to any URL
- Examples: links to external websites, social media

### Page
- Retrieve from the list of static pages already created
- Usually for the "About", "Contact", etc. menus

### Blog Category
- Take it from the blog category that has been created
- To display hubs/collections of articles per category

## Create a New Menu

1. Click the **Add New** button in the left panel
2. Fill in the name of the menu set (example: "Additional Menu")
3. Click **Save**
4. Start adding items to the new menu

## Tips

- **Don't be too complex** - Use a simple hierarchy (max 2-3 levels)
- **Logical sort** - Place important menu items at the beginning
- **Regular updates** - Remove menus that are no longer used
- **Responsive test** - Check the menu on the mobile device
