# Module

The **Module** page is used to activate or deactivate features available on the school website.

By organizing modules, you can customize your website to only display the features your school really needs.

## How to Access

1. Entered into [Admin Dashboard](../dashboard-admin.md)
2. Open menu **Admin → Appearance → Modules**

## Module Page Elements

This page displays a list of modules/features that can be enabled or disabled.

**Each module item usually contains:**
- **Module Name**: The name of the feature that can be enabled
- **Description**: A brief explanation of the module's function
- **Toggle/Switch**: Button to enable or disable the module

## Common Module Functions

Some modules that are usually available:

- **Blog** - Blog and news features
- **Gallery** - Photo/video gallery
- **Event** - School event calendar
- **Student Portal** - Student portal
- **LMS** - Learning Management System
- **Forum** - Online discussions
- And other modules according to the active package

## Activating Module

1. Find the module you want to activate in the list
2. Click the toggle/switch to activate
3. The module will automatically activate and its features will appear in the menu

## Disabling Modules

1. Find the module you want to disable
2. Click the toggle/switch to deactivate
3. The module will automatically be inactivated and the feature will disappear from the menu

:::warning
When disabling a module, all data associated with that module may remain stored but inaccessible. Make sure you really want to disable before doing so.
:::

## Tips

- **Start with basic modules** - Activate only those modules that are actually used
- **Don't have too many modules** - Fewer modules = faster website
- **Backup before changing** - If you are afraid of data loss, backup the system first
- **Team consultation** - Discuss with the IT team/school admin before changing important modules
