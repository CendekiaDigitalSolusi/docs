# Tenant Application Documentation

Welcome to the tenant application documentation page. This page will explain the various methods available in the Gradynex system, especially the tenant applications / applications that are accessed for your school.

Gradynex is a unified and integrated school management system. Gradynex provides school management systems, academic systems, LMS, and other supporting features.

## Key Features

- **User Dashboard** - Central access to all features according to your access rights
- **Admin Panel** - Comprehensive management for school administrators
- **User Management** - Manage user accounts and roles
- **Blog & News** - Manage school news content
- **Settings** - Configure the system as needed
