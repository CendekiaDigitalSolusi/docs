# User Dashboard

User Dashboard is the main page once you are successful [enter](../autentikasi/masuk.md). Here you can see summary information and quick access to key features according to your access rights.

## How to Access the Dashboard

1. Please [enter](../autentikasi/masuk.md) to the system
2. If successful, you will be automatically redirected to **User Dashboard**

## Dashboard Component

### Menu
The menu displays the available features according to the access rights (roles) of the account you are using.

### Brief Information
Displays important information and system status in an easy-to-understand visual format.

:::info
The menus and features that appear on your dashboard depend on your role/account permissions. Contact your school administrator if you need access to certain features.
:::

## Troubleshooting

### The dashboard is blank or data does not appear
- Reload page (press F5 or Cmd+R)
- Try logging out and logging in again
- Make sure your internet connection is stable
- Make sure your account is registered and active

### Can't see certain menus
- Menus may be hidden due to roles or access permissions
- Contact your school administrator to check your access

### Page loading is very slow
- Check your internet connection
- Repeat page refresh
- Contact an administrator if the problem persists
