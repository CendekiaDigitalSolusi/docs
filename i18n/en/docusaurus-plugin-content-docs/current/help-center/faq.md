## Troubleshooting

### I can't log in
Make sure:
1. Username and password are correct
2. Caps Lock is not active
3. Stable internet connection

If it still doesn't work, try resetting the password or contact the admin.

### The feature does not appear in the menu
Possibility:
1. You don't have access (check your role)
2. Feature has not been enabled in Module Settings
3. Contact the administrator to check

### Slow website
Try:
1. Reduce the number of articles per page
2. Compress images before uploading
3. Clear browser cache (Ctrl+F5)
4. Contact support if it remains slow
