# Troubleshooting

This page contains solutions to common problems you may encounter while using Gradynex.

## Login & Account

### Error: "Incorrect username or password"

**Solution:**
1. Make sure Caps Lock is not active
2. Check the spelling of username and password
3. Make sure there are no spaces before/after the username
4. Try using email as username if available
5. Reset password if it still doesn't work (see [Forgot Password](../central-app/autentikasi/lupa-kata-sandi.md))

### Error: "Email not verified"

**Solution:**
1. Check your email for a verification link
2. Check the spam/junk folder
3. Click the verification link received
4. If you don't receive the email, contact admin

### I can't create a new account

**Solution:**
1. Make sure you have access to register (if you need approval)
2. Fill in your personal data correctly and completely
3. Verify your email
4. Complete school data (if requested)
5. Contact admin if you still have problems

## Website & Appearance

### The website page is not loading

**Solution:**
1. Refresh page (F5 or Ctrl+R)
2. Clear browser cache (Ctrl+Shift+Delete or Cmd+Shift+Delete)
3. Try in another browser
4. Check your internet connection
5. Check server status (contact support)

### Images do not appear on the website

**Solution:**
1. Check the file format (must be JPG, PNG, or WebP)
2. Check the file size (don't make it too big > 5MB)
3. Re-upload image
4. Clear browser cache
5. Check file permissions (if you have server access)

### The menu does not appear correctly

**Solution:**
1. Check the menu structure in the Admin Panel
2. Make sure the menu item is active/published
3. Refresh the page
4. Clear browser cache
5. Check in another browser for verification

### The theme does not change once activated

**Solution:**
1. Hard refresh browser (Ctrl+F5 or Cmd+Shift+R)
2. Clear browser cache completely
3. Try in another browser
4. Check DNS connection (maybe provider's DNS cache)
5. Contact support if it still doesn't change

## Blog & Content

### My news does not appear on the blog page

**Solution:**
1. Make sure the news status is "Published" (not Draft)
2. Check the publication date (not in the future)
3. Make sure the category is selected
4. Refresh the page and clear cache
5. Check your account permissions

### Comments do not appear

**Solution:**
1. Check the discussion settings - are comments allowed?
2. Check whether the comment has been approved by the moderator
3. Refresh the page
4. Check if there are any filters that hide comments
5. Contact admin to check moderation

### Categories do not appear in the menu

**Solution:**
1. Make sure the category has been created
2. Check the menu structure - have categories been added?
3. Check category visibility/permissions
4. Refresh the page
5. Clear cache

## Emails & Notifications

### I didn't receive an email from the system

**Solution:**
1. Check your spam/junk email folder
2. Make sure your email is correct in your profile
3. Check the email server settings in the Admin Panel
4. Verify domain email sender
5. Contact support to check email logs

### Email goes to spam/junk

**Solution:**
1. Report non-spam to email provider
2. Add the sender's email to whitelist/contacts
3. Check SPF/DKIM records if you have DNS access
4. Contact support to improve deliverability

## Admin Panel

### "No data found" in the table

**Solution:**
1. Check the search filter - it may be too strict
2. Clear search filter and try again
3. Check the number of rows per page (maybe another page)
4. Make sure you have access to view this data
5. Try refreshing the page

### Action buttons don't work

**Solution:**
1. Refresh the page
2. Check your permissions/roles
3. Try in another browser
4. Clear browser cache
5. Contact admin

### Dashboard is empty / data does not appear

**Solution:**
1. Refresh page (F5)
2. Log out and re-login
3. Check internet connection
4. Clear browser cache
5. Check in another browser

## Performance / Speed

### Website is very slow

**Solution:**
1. Reduce the number of articles per page (Settings → Blog → Reading)
2. Compress/optimize images before uploading
3. Disable/disable unused modules
4. Clear cache systemhub ungi supports checking server logs
5. Upgrade the package if traffic is high

### Slow loading dashboard

**Solution:**
1. Clear browser cache
2. Clear cookies
3. Try incognito/private mode
4. Reduce complex filters/search
5. Contact support if it remains slow
