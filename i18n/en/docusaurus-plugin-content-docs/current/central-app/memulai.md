# Getting Started

To start using Gradynex quickly, follow the steps below.

## Setup Stages

### 1. Register
Create an account first through SSO Cendekia on the [Register](https://accounts.cendekiadigitalsolusi.co.id/register) page. The [full user guide is available here](/sso-cendekia/pengguna/daftar).

### 2. Sign In
After registering in SSO Cendekia, sign in to your account. You can sign in using email/password or social login (Google/Microsoft). See the [Sign In (Login) guide](./autentikasi/masuk.md).

### 3. Verify School Data
Complete your school profile data in the system. See the [School Data guide](./dasar/data-sekolah.md).

### 4. Subscribe
Choose the subscription plan that fits your school's needs. See the [Subscription guide](./dasar/berlangganan.md).

After completing these four stages, you are ready to use Gradynex.
