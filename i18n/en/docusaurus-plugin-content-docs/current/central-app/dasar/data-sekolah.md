# School Data

If you have just created an account via Scholar SSO, you are required to fill in your school data the first time you use Gradynex.

:::warning
Filling in this school data is **mandatory** for the verification process. If it is not filled in, you cannot subscribe to Gradynex.
:::

## Steps to Fill in School Data

1. After you [login](../autentikasi/masuk.md), the system will automatically redirect to the school data form page
2. Please fill in the form provided according to your school's information
3. Wait until it is verified by our admin team
4. If it has been verified, you can access all existing features including [subscribe](./berlangganan.md)

:::info
The verification process usually takes 1-2 working days.
:::
