# Data Sekolah

Jika Anda baru saja membuat akun atau baru melakukan registrasi, maka Anda wajib melakukan pengisian data sekolah ketika Anda pertama kali menggunakan akun ini. Pengisian data sekolah ini **wajib** untuk proses verifikasi, jika tidak diisi Anda tidak bisa berlangganan di Gradynex.

Untuk melakukan pengisian data sekolah ini sangat mudah:

1. Jika Anda sudah [login](../autentikasi/publish-your-docs.md), maka sistem akan otomatis mengarahkan ke halaman formulir data sekolah
2. Silahkan isi formulir yang tersedia sesuai asal sekolah Anda
3. Tunggu hingga diverifikasi oleh admin kami
4. Jika sudah diverifikasi, Anda dapat mengakses seluruh fitur yang ada di website termasuk berlangganan.
