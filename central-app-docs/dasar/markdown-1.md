# Trial

Untuk berlangganan Anda harus mempunyai akun dulu, jika belum punya bisa [registrasi](../autentikasi/quickstart.md) terlebih dahulu dan melakukan [verifikasi data sekolah](editor.md).

Jika akun sudah siap dan ingin berlangganan anda bisa mengikuti cara berikut ini:

1. [Masuk/Login](../autentikasi/publish-your-docs.md) ke Gradynex
2. Pilih menu **Langganan**, jika belum pernah berlangganan Anda berhak mendapatkan paket [trial](markdown-1.md) gratis 14 hari.
3. Klik **Coba Gratis** jika ingin mencoba paket trial kami.
4. Lalu isi form yang tersedia, dan klik submit.
5. Layanan trial akan otomatis aktif selama 14 hari.
