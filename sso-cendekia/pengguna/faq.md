# FAQ Pengguna

## Saya sudah login tapi kembali ke halaman login lagi

- Cek waktu/jam perangkat, pastikan akurat.
- Pastikan cookie browser tidak diblokir.
- Coba logout lalu login ulang.

## Akun saya terkunci

Biasanya terjadi karena percobaan login gagal berulang. Tunggu beberapa menit lalu coba lagi, atau reset kata sandi.

## Saya tidak menerima email verifikasi atau reset

- Cek folder Spam/Junk.
- Pastikan email benar.
- Jika masih tidak ada, hubungi admin untuk pengecekan layanan email.

## Saat klik Izinkan, muncul error redirect_uri

Ini biasanya masalah konfigurasi aplikasi (bukan kesalahan pengguna). Kirim screenshot error ke tim admin aplikasi.

## Kapan saya harus menghubungi admin?

Hubungi admin jika:

- akun terkunci berulang
- email verifikasi/reset tidak pernah masuk
- akses ditolak padahal akun aktif
- muncul error teknis setelah login
