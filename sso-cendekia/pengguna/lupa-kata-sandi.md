# Lupa Kata Sandi

Jika kamu lupa kata sandi, gunakan proses reset berikut.

1. Di halaman login, klik [Lupa Kata Sandi](https://accounts.cendekiadigitalsolusi.co.id/forgot-password).
2. Masukkan email akun.
3. Klik Kirim Link Reset.
4. Buka email dan klik link reset.
5. Masukkan kata sandi baru, lalu simpan.
6. Login kembali dengan kata sandi baru.

![Alur reset kata sandi](/img/sso-cendekia/reset-password-flow.svg)

## Catatan Penting

- Link reset hanya berlaku sementara.
- Jika link kedaluwarsa, ulangi proses Lupa Kata Sandi.
- Setelah reset, sesi login di perangkat lama bisa berakhir otomatis.
