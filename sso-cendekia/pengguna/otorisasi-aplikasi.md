# Persetujuan Akses Aplikasi (Consent)

Saat pertama kali menggunakan aplikasi tertentu, kamu mungkin melihat halaman persetujuan akses.

![Contoh layar consent](/img/sso-cendekia/consent-screen.svg)

## Artinya Apa?

Aplikasi meminta izin untuk mengakses data akunmu sesuai scope yang diminta.

Contoh data yang umum:

- `openid`: identitas unik pengguna
- `email`: alamat email
- `profile`: nama profil dasar

## Apa yang Harus Dicek Sebelum Klik Izinkan

- Nama aplikasi benar
- Domain aplikasi sesuai yang kamu kenal
- Jenis data yang diminta masuk akal

Jika ragu, klik Tolak lalu hubungi admin aplikasi.
