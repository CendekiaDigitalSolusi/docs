# Dokumentasi Aplikasi Tenant

Selamat datang dihalaman dokumentasi aplikasi tenant. Pada halaman ini akan menjelaskan berbagai cara yang ada di dalam sistem gradynex khususnya pada aplikasi tenant / aplikasi yang diakses untuk sekolah Anda.

Gradynex merupakan sebuah sistem manajemen sekolah terpadu dan terintegrasi. Gradynex menyediakan sistem manajemen sekolah, sistem akademik, LMS, dan fitur pendukung lainnya.
