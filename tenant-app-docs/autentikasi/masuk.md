# Masuk

Pastikan Anda sudah punya akun terlebih dahulu. Jika belum punya akun, Anda dapat menghubungi administrator sekolah Anda.&#x20;

1. Silahkan akses `namasekolah.sch.id/login` atau sesuaikan dengan domain sekolah Anda.
2. Masukkan username dan password Anda.
3. Jika data benar, Anda akan dialihkan ke dashboard pengguna.

{% hint style="info" %}
Jika Anda adalah administrator sekolah yang baru mengaktifkan layanan untuk pertama kali, username dan password Anda akan dikirim ke email yang terdaftar di sistem kami.
{% endhint %}
