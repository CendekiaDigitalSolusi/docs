# Tema

Halaman **Tema** dipakai untuk memilih tampilan website sekolah. Satu tema mengatur layout, komponen, dan gaya visual.

Kamu bisa:

* memilih tema yang tersedia
* mengganti tema aktif
* impor tema baru

{% hint style="warning" %}
Ganti tema bisa mengubah posisi menu, slider, dan tampilan halaman. Lakukan saat jam sepi dan cek halaman publik setelahnya.
{% endhint %}

***

### Tampilan halaman

Di halaman ini kamu akan melihat kartu-kartu tema.

Setiap kartu biasanya menampilkan:

* preview gambar tema
* nama tema (mis. _Blue Frost_, _School Vision_)
* menu opsi (ikon titik tiga) untuk aksi tambahan

Di kanan atas ada tombol **Impor Tema**.

***

### Memilih / mengaktifkan tema

Cara paling umum:

1. Tentukan tema yang ingin dipakai.
2. Klik menu **titik tiga** pada kartu tema.
3. Pilih aksi **Aktifkan / Gunakan** (nama tombol bisa berbeda).
4. Buka halaman publik untuk memastikan tampilan sudah berubah.

{% hint style="info" %}
Jika tema terlihat tidak berubah, coba hard refresh di browser (Ctrl+F5).
{% endhint %}

***

### Impor tema baru

Gunakan tombol **Impor Tema** jika kamu menerima file tema dari tim pengembang.

{% stepper %}
{% step %}
#### 1) Klik “Impor Tema”

Tombol berada di kanan atas.
{% endstep %}

{% step %}
#### 2) Pilih file tema

Pilih file yang disediakan (biasanya format khusus dari sistem).
{% endstep %}

{% step %}
#### 3) Selesaikan impor

Setelah berhasil, tema baru akan muncul sebagai kartu.
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
Jika impor gagal, cek ukuran file dan koneksi internet. Jika masih gagal, minta ulang paket tema yang benar.
{% endhint %}

***

### Mengelola tema (opsi di kartu)

Di menu **titik tiga** pada setiap tema, biasanya ada opsi seperti:

* **Preview**: melihat contoh tampilan.
* **Aktifkan / Gunakan**: menjadikan tema sebagai tema aktif.
* **Hapus**: menghapus tema dari daftar (biasanya untuk tema impor).

{% hint style="danger" %}
Jangan hapus tema default jika tidak yakin. Simpan minimal satu tema cadangan untuk rollback.
{% endhint %}

***

* [ ] Header & footer tampil normal.
* [ ] Menu navigasi berfungsi.
* [ ] Slider tampil (jika dipakai). Lihat: [Slider Gambar](../blog/slider-gambar.md)
* [ ] Halaman penting (Profil/Kontak) tidak rusak. Lihat: [Halaman](../blog/halaman.md)
* [ ] Berita masih terbaca dan gambar tidak pecah. Lihat: [Berita](../blog/berita.md)

### Troubleshooting

#### Tema tidak berubah setelah diaktifkan

* Refresh halaman publik.
* Coba hard refresh (Ctrl+F5).
* Coba buka di mode incognito.

#### Tampilan jadi berantakan

* Kembali ke halaman **Tema**.
* Aktifkan tema sebelumnya (rollback).
* Cek konfigurasi menu di [Menu](menu.md).

#### Tidak ada tombol Impor Tema

Kemungkinan akun kamu tidak punya izin. Cek di [Hak Akses](../peran.md).
