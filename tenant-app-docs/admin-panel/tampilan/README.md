# Tampilan

Bagian **Tampilan** mengatur apa yang dilihat pengunjung di website sekolah. Fokusnya ada dua: **navigasi** dan **tema**.

### Quick start

Urutan yang paling aman saat setup tampilan:

{% stepper %}
{% step %}
#### 1) Pilih tema

Pilih tema yang paling mendekati kebutuhan sekolah, lalu aktifkan.

Lihat: [Tema](tema.md)
{% endstep %}

{% step %}
#### 2) Susun menu

Atur menu header/footer agar pengunjung mudah menemukan halaman penting.

Lihat: [Menu](menu.md)
{% endstep %}
{% endstepper %}
