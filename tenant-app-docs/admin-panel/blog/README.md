# Blog

Modul **Blog** berisi fitur untuk mengelola konten publik sekolah: berita, halaman statis, slider, dan moderasi komentar.

### Mulai dari mana?

Kalau baru setup, urutan yang paling aman:

{% stepper %}
{% step %}
#### 1) Siapkan struktur konten

* Buat [Kategori](kategori.md) untuk pengelompokan utama.
* Buat [Tag](tag.md) untuk label topik.
{% endstep %}

{% step %}
#### 2) Tulis konten

* Tulis [Berita](berita.md) untuk konten kronologis.
* Buat [Halaman](halaman.md) untuk konten statis.
{% endstep %}

{% step %}
#### 3) Percantik tampilan

Kelola [Slider Gambar](slider-gambar.md) untuk banner/hero di beranda.
{% endstep %}

{% step %}
#### 4) Jaga diskusi tetap rapi

Moderasi [Komentar](komentar.md) untuk menghapus spam dan memastikan etika.
{% endstep %}
{% endstepper %}
