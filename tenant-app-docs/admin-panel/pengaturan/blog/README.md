# Blog

Bagian **Pengaturan Blog** mengatur default perilaku blog sekolah. Di sini kamu mengatur dua hal utama:

* **Membaca**: jumlah artikel yang tampil dan panjang ringkasan.
* **Diskusi**: aturan komentar untuk artikel baru.

### Rekomendasi cepat

Kalau kamu butuh konfigurasi yang aman untuk sekolah:

* Membaca: 5–10 berita per halaman, ringkasan 120–180 karakter.
* Diskusi: wajib login + tutup komentar artikel lama (mis. 14–30 hari).

### Catatan penting

{% hint style="warning" %}
Perubahan di **Pengaturan Diskusi** biasanya hanya berlaku untuk **artikel baru**. Artikel lama bisa tetap memakai setting lamanya.
{% endhint %}

{% hint style="info" %}
Untuk moderasi komentar yang sudah masuk, gunakan menu [Komentar](../../blog/komentar.md).
{% endhint %}
