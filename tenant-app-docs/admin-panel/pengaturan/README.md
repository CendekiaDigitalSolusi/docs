# Pengaturan

Bagian **Pengaturan** mengatur konfigurasi inti aplikasi dan website sekolah. Isinya campuran: identitas sekolah, email sistem, sampai default perilaku blog.

### Quick setup (urutannya enak)

{% stepper %}
{% step %}
#### 1) Lengkapi identitas sekolah

Isi data sekolah dulu. Ini dipakai di banyak tempat.

Lihat: [Profil Sekolah](profil-sekolah.md)
{% endstep %}

{% step %}
#### 2) Atur tampilan dasar website

Meta SEO, logo, favicon, zona waktu, dan reCAPTCHA.

Lihat: [Pengaturan Umum](pengaturan-umum.md)
{% endstep %}

{% step %}
#### 3) Konfigurasikan email sistem

Wajib kalau kamu ingin fitur seperti reset password berjalan.

Lihat: [Pengaturan Server Email](pengaturan-server-email.md)
{% endstep %}

{% step %}
#### 4) Rapikan default blog

Atur jumlah artikel per halaman dan aturan komentar.

Lihat: [Pengaturan Blog](blog/)
{% endstep %}
{% endstepper %}
