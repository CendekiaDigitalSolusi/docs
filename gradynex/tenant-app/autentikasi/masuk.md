# Cara Masuk (Login)

Untuk mengakses aplikasi tenant Gradynex, Anda perlu memiliki akun yang sudah terdaftar. Jika belum punya akun, hubungi administrator sekolah Anda.

## Langkah-Langkah Masuk

1. Buka halaman login di domain sekolah Anda (contoh: `namasekolah.sch.id/login`).
2. Masukkan username dan password Anda.
3. Klik tombol **Masuk**.
4. Jika data benar, Anda akan masuk ke [dashboard](../user-panel/dashboard.md).

:::info
**Untuk administrator baru:** Username dan password Anda akan dikirim ke email yang terdaftar di sistem kami saat layanan diaktifkan untuk pertama kali.
:::

## Jika Gagal Login?

- Pastikan username dan password benar.
- Cek apakah Caps Lock keyboard aktif.
- Gunakan menu **Lupa Kata Sandi** jika lupa password.
- Hubungi administrator sekolah untuk bantuan lebih lanjut.
