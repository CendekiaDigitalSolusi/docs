# Masuk

Pastikan Anda sudah punya akun terlebih dahulu. Jika belum punya akun, silakan hubungi administrator sekolah Anda.

## Langkah-Langkah Masuk

1. Akses halaman login dengan URL: `namasekolah.sch.id/login` (sesuaikan dengan domain sekolah Anda)
2. Masukkan username dan password Anda
3. Klik tombol **Masuk**
4. Jika data benar, Anda akan dialihkan ke [dashboard pengguna](../user-panel/dashboard.md)

:::info
**Untuk administrator baru:** Username dan password Anda akan dikirim ke email yang terdaftar di sistem kami saat layanan diaktifkan untuk pertama kali.
:::

## Catatan

- Jaga kerahasiaan username dan password Anda
- Jangan gunakan komputer umum untuk login jika memungkinkan
- Selalu logout setelah selesai menggunakan sistem
