# Lupa Kata Sandi

Jika Anda lupa password akun tenant Gradynex, gunakan proses reset berikut.

## Langkah-Langkah Reset Kata Sandi

1. Buka halaman login di domain sekolah Anda (contoh: `namasekolah.sch.id/login`).
2. Klik link **Lupa Kata Sandi**.
3. Masukkan email akun Anda.
4. Klik tombol **Kirim Link Reset**.
5. Buka email (cek folder Spam jika tidak ada).
6. Klik link reset dari email Gradynex.
7. Masukkan password baru dan konfirmasi password.
8. Klik **Ganti Kata Sandi**.
9. Login kembali dengan password baru.

## Catatan Penting

- Link reset hanya berlaku sementara.
- Jika link kedaluwarsa, ulangi proses dari awal.
- Setelah reset, sesi login di perangkat lain mungkin berakhir.
- Jika email tidak diterima, cek folder Spam atau hubungi administrator.
