# Profil Sekolah

Halaman **Profil Sekolah** menyimpan data identitas sekolah yang digunakan di berbagai tempat dalam sistem.

Data ini biasanya muncul di:
- Website publik
- Halaman profil resmi
- Footer website
- Metadata dan integrasi

:::warning
**Penting:** Isi data ini dengan benar sejak awal. Beberapa data (seperti NPSN dan alamat) sering dipakai untuk administrasi dan integrasi dengan sistem lainnya.
:::

## Cara Mengakses

1. Masuk ke [Dashboard Admin](../dashboard-admin.md)
2. Buka menu **Admin → Pengaturan → Profil Sekolah**

## Alur Pengisian yang Disarankan

Jika Anda baru pertama kali setup, ikuti urutan ini agar dropdown alamat tidak membingungkan:

### 1. Isi Identitas Dasar
- **Nama Sekolah**: Nama resmi sekolah Anda
- **NPSN**: Nomor Pokok Sekolah Nasional

### 2. Isi Alamat Bertahap
Pilih secara bertahap:
1. **Provinsi**
2. **Kota/Kabupaten**
3. **Kecamatan**
4. **Kelurahan/Desa**
5. **Alamat Lengkap**: Isi alamat detail jalan/no. rumah

### 3. Lengkapi Profil Publik
- **Nama Kepala Sekolah**: Nama pimpinan sekolah
- **Pesan Sambutan**: Sambutan dari kepala sekolah (opsional)
- **Deskripsi Sekolah**: Deskripsi singkat atau motto sekolah
- **URL Profil**: Link ke profil resmi/website sekolah (opsional)

### 4. Isi Statistik & Media Sosial
- **Jumlah Guru**: Total guru di sekolah
- **Jumlah Siswa**: Total siswa di sekolah
- **Jumlah Staf**: Total staf/karyawan
- **Jumlah Alumni**: Total alumni (opsional)
- **Media Sosial**: Tautan ke akun resmi (Facebook, Instagram, YouTube, dll)

### 5. Simpan
- Scroll ke bawah halaman
- Klik tombol **Simpan** untuk menyimpan perubahan

## Tips

- **Data akurat** - Pastikan semua data sesuai dengan data resmi sekolah
- **Photo berkualitas** - Gunakan foto kepala sekolah dan visual sekolah berkualitas tinggi
- **Update berkala** - Update jumlah siswa, guru, dan informasi lainnya setiap tahun ajaran baru
- **Media sosial aktif** - Pastikan tautan media sosial mengarah ke akun resmi dan aktif
- **NPSN valid** - Verifikasi NPSN Anda di website Kemendikbud sebelum diisi
