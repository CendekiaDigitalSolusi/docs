# Pengaturan

Modul Pengaturan memungkinkan Anda mengkonfigurasi sistem sesuai kebutuhan sekolah Anda.

## Komponen Pengaturan

Modul ini mencakup:

- **[Pengaturan Umum](./pengaturan-umum.md)** - Konfigurasi dasar website dan SEO
- **[Profil Sekolah](./profil-sekolah.md)** - Data identitas dan profil sekolah
- **[Pengaturan Server Email](./pengaturan-server-email.md)** - Konfigurasi email otomatis
- **Blog** - Pengaturan khusus untuk modul blog
  - [Pengaturan Membaca](./blog/pengaturan-membaca.md)
  - [Pengaturan Diskusi](./blog/pengaturan-diskusi.md)

## Panduan Cepat

Untuk mengatur sistem, pelajari topik-topik di atas sesuai kebutuhan Anda.

### Untuk Setup Awal
1. Mulai dengan [Pengaturan Umum](./pengaturan-umum.md)
2. Lanjut ke [Profil Sekolah](./profil-sekolah.md)
3. Setup [Email Server](./pengaturan-server-email.md) jika perlu

### Untuk Konfigurasi Blog
- Lihat [Pengaturan Blog](./blog/README.md)
