# Pengaturan Blog

Modul ini mengatur konfigurasi untuk fitur blog di website sekolah.

## Komponen Pengaturan Blog

- **[Pengaturan Membaca](./pengaturan-membaca.md)** - Mengatur tampilan daftar artikel
- **[Pengaturan Diskusi](./pengaturan-diskusi.md)** - Mengatur fitur komentar

## Panduan Cepat

Untuk mengkonfigurasi blog, pelajari topik-topik di atas sesuai kebutuhan Anda.

### Untuk Mengatur Tampilan Blog
- Lihat panduan [Pengaturan Membaca](./pengaturan-membaca.md)

### Untuk Mengatur Komentar & Moderasi
- Lihat panduan [Pengaturan Diskusi](./pengaturan-diskusi.md)
