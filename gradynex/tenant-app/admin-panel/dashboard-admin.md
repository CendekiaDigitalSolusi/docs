# Dashboard Admin

Dashboard Admin adalah halaman utama di admin panel. Di sini Anda dapat melihat ringkasan informasi penting dan akses cepat ke fitur pengelolaan sekolah.

## Cara Mengakses Dashboard Admin

1. [Masuk](../autentikasi/masuk.md) ke sistem dengan akun administrator
2. Anda akan melihat dashboard pengguna
3. Klik menu **Admin Panel** untuk masuk ke dashboard admin
4. Sekarang Anda berada di Dashboard Admin

:::info
**Anda tidak melihat menu Admin Panel?** Berarti akun Anda tidak memiliki akses ke Admin Panel. Hubungi administrator utama sekolah untuk meminta akses.
:::

## Komponen Dashboard Admin

### Menu
Daftar fitur yang dapat diakses sesuai dengan role/hak akses Anda, meliputi:
- Pengelolaan Pengguna
- Pengaturan Sistem
- Manajemen Blog
- Tampilan/Tema
- Dan fitur lainnya sesuai modul yang aktif

### Ringkasan Data
Kartu yang menampilkan informasi statistik dan status penting sistem sekolah Anda.

:::warning
Fitur pada dashboard admin dapat berubah sesuai dengan modul yang diaktifkan atau versi sistem yang Anda gunakan.
:::
