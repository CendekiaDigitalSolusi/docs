# Blog & Berita

Modul Blog memungkinkan Anda untuk mengelola dan menerbitkan konten ke halaman publik website sekolah.

## Komponen Blog

Modul blog terdiri dari beberapa bagian yang dapat Anda kelola:

- **[Berita](./berita.md)** - Konten dinamis berbasis tanggal
- **[Halaman](./halaman.md)** - Konten statis evergreen
- **[Kategori](./kategori.md)** - Pengelompokan konten
- **[Tag](./tag.md)** - Label tambahan untuk konten
- **[Slider Gambar](./slider-gambar.md)** - Banner/carousel di homepage
- **[Komentar](./komentar.md)** - Moderasi komentar pembaca

## Panduan Cepat

Untuk memulai membuat konten, pelajari topik-topik di atas sesuai dengan jenis konten yang ingin Anda buat.

- Lihat panduan [Berita](./berita.md)

### Untuk Konten Statis
- Lihat panduan [Halaman](./halaman.md)
