# Pengguna

Halaman **Pengguna** digunakan untuk mengelola akun yang dapat masuk ke sistem Tenant.

Di sini Anda dapat:
- Menambah akun pengguna baru
- Mengatur role/peran pengguna
- Mencari akun tertentu dengan cepat
- Mengubah data pengguna
- Menghapus akun

## Cara Mengakses

1. Masuk ke [Dashboard Admin](./dashboard-admin.md)
2. Buka menu **Admin → Pengguna**

## Elemen Halaman Pengguna

### Toolbar di Atas Tabel

- **Tombol Tambah Baru**: Membuat akun pengguna baru
- **Dropdown Aksi**: Melakukan aksi massal untuk item yang dicentang
- **Dropdown Kolom**: Mengatur kolom mana saja yang ditampilkan
- **Dropdown Jumlah Baris**: Mengatur berapa banyak data yang ditampilkan per halaman (contoh: 10, 25, 50)

### Tabel Pengguna

Tabel menampilkan kolom-kolom berikut:

| Kolom | Keterangan |
|-------|-----------|
| **Nama** | Nama lengkap pengguna |
| **Username** | Nama pengguna untuk login |
| **Email** | Email pengguna |
| **Peran** | Role/hak akses pengguna |
| **Aksi** | Tombol Edit/Hapus untuk setiap pengguna |

### Filter Cepat

Setiap header kolom memiliki filter untuk pencarian cepat:
- Filter **Nama**
- Filter **Username**
- Filter **Email**
- Filter **Peran** (dropdown)

## Menambah Pengguna Baru

1. Klik tombol **Tambah Baru**
2. Isi formulir dengan data pengguna:
   - Nama lengkap
   - Username (untuk login)
   - Email
   - Password
   - Konfirmasi password
   - Pilih Peran/Role
3. Klik **Simpan**

## Mengubah Data Pengguna

1. Cari pengguna yang ingin diubah menggunakan filter atau pencarian
2. Klik tombol **Edit** (ikon pensil) pada baris pengguna
3. Ubah data yang diperlukan
4. Klik **Simpan**

## Menghapus Pengguna

1. Cari pengguna yang ingin dihapus
2. Klik tombol **Hapus** (ikon sampah) pada baris pengguna
3. Konfirmasi penghapusan

:::warning
Hati-hati saat menghapus pengguna, karena tindakan ini tidak dapat dibatalkan. Pastikan pengguna tersebut tidak lagi diperlukan.
:::
