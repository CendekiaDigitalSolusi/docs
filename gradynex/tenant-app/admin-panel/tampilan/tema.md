# Tema

Halaman **Tema** digunakan untuk memilih tampilan/desain website sekolah. Satu tema mengatur layout, komponen visual, warna, dan gaya keseluruhan website.

## Fitur Tema

Dengan halaman ini, Anda dapat:
- Memilih tema dari daftar tema yang tersedia
- Mengganti tema aktif
- Mengimpor tema baru

:::warning
**Perhatian:** Mengganti tema dapat mengubah posisi menu, slider, dan tampilan halaman. Lakukan perubahan saat jam-jam sepi dan periksa halaman publik setelahnya untuk memastikan semua tampak baik.
:::

## Cara Mengakses

1. Masuk ke [Dashboard Admin](../dashboard-admin.md)
2. Buka menu **Admin → Tampilan → Tema**

## Elemen Halaman Tema

Halaman ini menampilkan kartu-kartu tema yang tersedia.

**Setiap kartu tema biasanya menampilkan:**
- **Preview gambar** - Tampilan pratinjau dari tema
- **Nama Tema** - Nama tema (contoh: "Blue Frost", "School Vision")
- **Menu Opsi** - Ikon titik tiga untuk aksi tambahan

**Di kanan atas ada tombol:**
- **Impor Tema** - Untuk mengimpor tema baru dari file

## Memilih/Mengaktifkan Tema

Cara paling umum untuk mengganti tema:

1. Tentukan tema yang ingin dipakai dari daftar yang tersedia
2. Klik menu **titik tiga** pada kartu tema tersebut
3. Pilih opsi **Aktifkan** atau **Gunakan** (nama tombol mungkin berbeda)
4. Buka halaman publik website untuk memastikan tampilan sudah berubah

:::info
Jika tema terlihat tidak berubah setelah aktivasi, coba hard refresh di browser dengan menekan **Ctrl+F5** (Windows) atau **Cmd+Shift+R** (Mac).
:::

## Mengimpor Tema Baru

Jika Anda memiliki file tema baru:

1. Klik tombol **Impor Tema** di kanan atas
2. Pilih file tema yang ingin diimpor
3. Tunggu proses impor selesai
4. Tema baru akan muncul di daftar tema

## Tips

- **Test di staging dulu** - Jika ada lingkungan testing, coba tema baru di sana terlebih dahulu
- **Backup tema lama** - Sebelum menghapus tema, pastikan Anda punya backup jika membutuhkannya lagi
- **Cek responsive** - Buka tema baru di berbagai perangkat (desktop, tablet, mobile)
- **Minimalkan custom CSS** - Jika perlu custom CSS, dokumentasikan untuk kemudahan maintenance

## Troubleshooting

**Tema tidak berubah setelah diaktifkan:**
- Coba hard refresh browser (Ctrl+F5)
- Clear cache browser
- Coba di browser lain

**Tema tidak tampil sempurna:**
- Cek kompatibilitas versi tema dengan versi sistem
- Lihat console browser untuk error message
- Hubungi support jika tema rusak
