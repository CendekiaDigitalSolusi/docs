# Tampilan / Theme

Modul Tampilan memungkinkan Anda mengatur visual dan layout website sekolah.

## Komponen Tampilan

Modul ini mencakup:

- **[Menu](./menu.md)** - Mengelola navigasi website
- **[Tema](./tema.md)** - Memilih dan mengubah tampilan website
- **[Modul](./modul.md)** - Mengaktifkan/menonaktifkan fitur

## Panduan Cepat

Untuk mengubah tampilan website, pelajari topik-topik di atas sesuai kebutuhan Anda.

### Untuk Mengubah Layout Menu
- Lihat panduan [Menu](./menu.md)

### Untuk Mengganti Tema/Warna Website
- Lihat panduan [Tema](./tema.md)

### Untuk Mengelola Fitur yang Ditampilkan
- Lihat panduan [Modul](./modul.md)
