# Menu

Halaman **Menu** digunakan untuk mengatur navigasi website sekolah. Anda dapat mengelola menu untuk **Header** dan **Footer** (atau varian lain yang tersedia).

Pikirkan halaman ini sebagai "peta jalan" untuk pengunjung website Anda.

## Cara Mengakses

1. Masuk ke [Dashboard Admin](../dashboard-admin.md)
2. Buka menu **Admin → Tampilan → Menu**

## Mode Utama

### Mode 1: Struktur Menu
Tab **Struktur Menu** digunakan untuk mengatur urutan dan hierarki item menu.

### Mode 2: Kelola Menu
Tab **Kelola Menu** digunakan untuk mengelola daftar item menu (buat/edit/hapus).

:::info
**Urutan Kerja:** Biasanya Anda membuat item menu dulu di **Kelola Menu**, lalu menyusun posisinya di **Struktur Menu**.
:::

## Komponen Halaman Menu

### Panel Kiri: Pilih Menu

Di panel kiri, pilih set menu yang akan diedit. Menu yang sering tersedia:

- **Menu Header** - Navigasi utama di bagian atas website
- **Menu Footer** - Link di bagian bawah website

**Tombol di panel kiri:**
- **Tambah Baru**: Membuat set menu baru
- **Ubah**: Mengganti nama/konfigurasi menu

### Panel Kanan: Kelola Struktur & Item

Bergantung pada tab yang dipilih, Anda dapat:

**Di Tab Kelola Menu:**
- Membuat item menu baru
- Mengedit item menu yang ada
- Menghapus item menu

**Di Tab Struktur Menu:**
- Mengatur urutan item (drag & drop)
- Menyusun hierarki/submenu
- Menyembunyikan item tertentu

## Menambah Item Menu

Item menu dapat berasal dari beberapa sumber:

### Tautan Manual
- Link manual ke URL apapun
- Contoh: link ke website eksternal, social media

### Halaman
- Ambil dari daftar halaman statis yang sudah dibuat
- Biasanya untuk menu "Tentang", "Kontak", dll

### Kategori Blog
- Ambil dari kategori blog yang sudah dibuat
- Untuk menampilkan hub/kumpulan artikel per kategori

## Membuat Menu Baru

1. Klik tombol **Tambah Baru** di panel kiri
2. Isi nama set menu (contoh: "Menu Tambahan")
3. Klik **Simpan**
4. Mulai tambahkan item ke menu baru tersebut

## Tips

- **Jangan terlalu kompleks** - Gunakan hierarki yang sederhana (max 2-3 level)
- **Urutkan logis** - Tempat item menu yang penting di awal
- **Update teratur** - Hapus menu yang sudah tidak dipakai
- **Test responsif** - Cek menu di perangkat mobile
