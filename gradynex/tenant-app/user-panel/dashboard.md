# Dashboard Pengguna

Dashboard Pengguna adalah halaman utama setelah Anda berhasil [masuk](../autentikasi/masuk.md). Di sini Anda dapat melihat ringkasan informasi dan akses cepat ke fitur utama sesuai dengan hak akses Anda.

## Cara Mengakses Dashboard

1. Silakan [masuk](../autentikasi/masuk.md) ke sistem
2. Jika berhasil, Anda akan dialihkan otomatis ke **Dashboard Pengguna**

## Komponen Dashboard

### Menu
Menu menampilkan fitur-fitur yang tersedia sesuai dengan hak akses (role) dari akun yang Anda gunakan.

### Informasi Ringkas
Menampilkan informasi penting dan status sistem dalam format visual yang mudah dipahami.

:::info
Menu dan fitur yang tampil di dashboard Anda tergantung pada role/izin akun Anda. Hubungi administrator sekolah jika Anda memerlukan akses ke fitur tertentu.
:::

## Troubleshooting

### Dashboard kosong atau data tidak muncul
- Muat ulang halaman (tekan F5 atau Cmd+R)
- Coba logout dan masuk ulang
- Pastikan koneksi internet Anda stabil
- Pastikan akun Anda sudah terdaftar dan aktif

### Tidak bisa melihat menu tertentu
- Menu mungkin disembunyikan karena role atau izin akses
- Hubungi administrator sekolah untuk pengecekan akses Anda

### Halaman loading sangat lambat
- Cek koneksi internet Anda
- Ulangi refresh halaman
- Hubungi administrator jika masalah berlanjut
