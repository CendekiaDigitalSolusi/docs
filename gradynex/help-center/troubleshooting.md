# Troubleshooting

Halaman ini berisi solusi untuk masalah umum yang mungkin Anda temui saat menggunakan Gradynex.

## Login & Akun

### Error: "Username atau password salah"

**Solusi:**
1. Pastikan Caps Lock tidak aktif
2. Periksa ejaan username dan password
3. Pastikan tidak ada spasi sebelum/sesudah username
4. Coba gunakan email sebagai username jika tersedia
5. Reset password jika masih tidak bisa (lihat [Lupa Kata Sandi](../central-app/autentikasi/lupa-kata-sandi.md))

### Error: "Email belum diverifikasi"

**Solusi:**
1. Periksa email untuk link verifikasi
2. Cek folder spam/junk
3. Klik link verifikasi yang diterima
4. Jika tidak menerima email, hubungi admin

### Saya tidak bisa membuat akun baru

**Solusi:**
1. Pastikan Anda sudah punya akses untuk registrasi (jika perlu approval)
2. Isi data diri dengan benar dan lengkap
3. Verifikasi email Anda
4. Melengkapi data sekolah (jika diminta)
5. Hubungi admin jika masih bermasalah

## Website & Tampilan

### Halaman website tidak loading

**Solusi:**
1. Refresh halaman (F5 atau Ctrl+R)
2. Clear cache browser (Ctrl+Shift+Delete atau Cmd+Shift+Delete)
3. Coba di browser lain
4. Periksa koneksi internet Anda
5. Cek status server (hubungi support)

### Gambar tidak tampil di website

**Solusi:**
1. Periksa format file (harus JPG, PNG, atau WebP)
2. Cek ukuran file (jangan terlalu besar >5MB)
3. Upload ulang gambar
4. Clear cache browser
5. Periksa permission file (jika punya akses server)

### Menu tidak muncul dengan benar

**Solusi:**
1. Periksa struktur menu di Admin Panel
2. Pastikan item menu sudah aktif/publish
3. Refresh halaman
4. Clear cache browser
5. Periksa di browser lain untuk verifikasi

### Tema tidak berubah setelah diaktifkan

**Solusi:**
1. Hard refresh browser (Ctrl+F5 atau Cmd+Shift+R)
2. Clear cache browser sepenuhnya
3. Coba di browser lain
4. Cek koneksi DNS (mungkin cache DNS provider)
5. Hubungi support jika tetap tidak berubah

## Blog & Konten

### Berita saya tidak tampil di halaman blog

**Solusi:**
1. Pastikan status berita adalah "Published" (bukan Draft)
2. Cek tanggal publikasi (jangan di masa depan)
3. Pastikan kategori sudah dipilih
4. Refresh halaman dan clear cache
5. Cek permission akun Anda

### Komentar tidak tampil

**Solusi:**
1. Periksa settings diskusi - apakah komentar diizinkan?
2. Cek apakah komentar sudah disetujui oleh moderator
3. Refresh halaman
4. Periksa apakah ada filter yang menyembunyikan komentar
5. Hubungi admin untuk cek moderasi

### Kategori tidak muncul di menu

**Solusi:**
1. Pastikan kategori sudah dibuat
2. Periksa struktur menu - apakah kategori sudah ditambahkan?
3. Cek visibility/permission kategori
4. Refresh halaman
5. Clear cache

## Email & Notifikasi

### Saya tidak menerima email dari sistem

**Solusi:**
1. Periksa folder spam/junk email
2. Pastikan email Anda benar di profil
3. Cek setting server email di Admin Panel
4. Verifikasi sender email domain
5. Hubungi support untuk check email logs

### Email masuk ke spam/junk

**Solusi:**
1. Lapor bukan spam ke email provider
2. Tambahkan email pengirim ke whitelist/contacts
3. Cek SPF/DKIM record jika punya akses DNS
4. Hubungi support untuk improve deliverability

## Admin Panel

### "Tidak ada data yang ditemukan" di tabel

**Solusi:**
1. Periksa filter pencarian - mungkin terlalu ketat
2. Clear search filter dan coba lagi
3. Cek jumlah baris per halaman (mungkin halaman lain)
4. Pastikan Anda punya akses untuk melihat data ini
5. Coba refresh halaman

### Tombol aksi tidak berfungsi

**Solusi:**
1. Refresh halaman
2. Cek permission/role Anda
3. Coba di browser lain
4. Clear cache browser
5. Hubungi admin

### Dashboard kosong / data tidak muncul

**Solusi:**
1. Refresh halaman (F5)
2. Logout dan login ulang
3. Periksa koneksi internet
4. Clear cache browser
5. Cek di browser lain

## Performance / Kecepatan

### Website sangat lambat

**Solusi:**
1. Kurangi jumlah artikel per halaman (Pengaturan → Blog → Membaca)
2. Compress/optimize gambar sebelum upload
3. Disable/nonaktifkan modul yang tidak digunakan
4. Clear cache sistemhub ungi support untuk check server logs
5. Upgrade paket jika traffic tinggi

### Dashboard loading lambat

**Solusi:**
1. Clear browser cache
2. Clear cookie
3. Try incognito/private mode
4. Kurangi filter/search yang kompleks
5. Hubungi support jika tetap lambat
