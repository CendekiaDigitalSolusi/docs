# Data Sekolah

Jika Anda baru saja membuat akun atau baru melakukan registrasi, maka Anda wajib melakukan pengisian data sekolah ketika Anda pertama kali menggunakan akun ini.

:::warning
Pengisian data sekolah ini **wajib** untuk proses verifikasi. Jika tidak diisi, Anda tidak bisa berlangganan di Gradynex.
:::

## Langkah-Langkah Pengisian Data Sekolah

1. Setelah Anda [login](../autentikasi/masuk.md), sistem akan secara otomatis mengarahkan ke halaman formulir data sekolah
2. Silahkan isi formulir yang tersedia sesuai dengan informasi asal sekolah Anda
3. Tunggu hingga diverifikasi oleh tim admin kami
4. Jika sudah diverifikasi, Anda dapat mengakses seluruh fitur yang ada termasuk [berlangganan](./berlangganan.md)

:::info
Proses verifikasi biasanya memakan waktu 1-2 hari kerja.
:::
